# wp-tdd
A Test-Driven Development WordPress plugin starter

## Setup
1. **Required:** Move `pre-commit` file to `.git/hooks/` folder
2. **Required:** Run `composer install`
3. **Optional:** Change `Text Domain` from `phpcs.xml > text_domain > element`
